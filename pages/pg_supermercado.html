SISTEMA PARA SUPERMERCADO

Gerenciar seu supermercado ficou mais simples com sistema de automação comercial desenvolvido pela D2D.


SUPERMERCADO ORGANIZADO


A D2D oferece um sistema completo para supermercados, com uma frente de Caixa (PDV) moderna, intuitiva e de altíssima produtividade, homologado para Emissor de Cupom Fiscal, Nota Fiscal Paulista, Nota Fiscal Eletrônica e TEF.

Com o sistema D2D, é possível controlar todo o seu estoque, vendas diárias, fechamento de caixa por chexk-out, Sangria/Suprimento de caixa, integrado com balanças Filizola/Toledo, terminal de consulta de Preços - Gertec, Gaveta Automática de dinheiro, impressão de etiquetas de Gondulas.

Sistema homologado

Sistema homologado para diversas impressoras fiscais. Software integrado com Nfe (Nota fiscal eletrônica), NfCe ( Nota fiscal consumidor eletrônica), TEF (Transferência eletrônica de fundos) e SAT(Cupom Fiscal Eletrônico).

Terminal

Fácil de usar

Gaveta automática

Controle financeiro

Controle do fluxo de caixa com geração diária de relatório, com dados de abertura, fechamento e quebra de caixa.

Leitor de código de barra

Agilidade na venda

O ponto de venda está integrado com todo o sistema, na finalização de vendas, o sistema da baixa no estoque dos produtos vendidos e gera relatórios das vendas, do caixa, de cada produto vendido.
